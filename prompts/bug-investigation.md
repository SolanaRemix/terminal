# AI Prompt: Bug Investigation

You are debugging an issue in the Terminal repository. Analyze the problem systematically and propose solutions.

## Investigation Process

### 1. Understand the Issue
- What is the expected behavior?
- What is the actual behavior?
- When did this start happening?
- Can you reproduce it consistently?

### 2. Gather Information
- Error messages and stack traces
- Logs and console output
- Recent code changes
- Environment configuration

### 3. Isolate the Problem
- Which component is affected?
- Which file/function contains the bug?
- What are the inputs that trigger it?
- Does it affect all users or specific cases?

### 4. Analyze Root Cause
- What is the underlying cause?
- Why did this issue occur?
- Are there related issues?

### 5. Propose Solutions
- Quick fix (immediate workaround)
- Proper fix (long-term solution)
- Prevention (how to avoid similar issues)

## Output Format

**Issue Summary**: Brief description

**Root Cause**: Explanation of what went wrong

**Reproduction Steps**:
1. Step 1
2. Step 2
3. Step 3

**Proposed Fix**:
```typescript
// Code changes with comments
```

**Testing Plan**:
- Test case 1
- Test case 2

**Prevention Measures**:
- How to prevent this in the future
