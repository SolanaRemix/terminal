# AI Prompt Templates

This directory contains AI prompt templates for the CyberAi ecosystem integration.

## Available Prompts

### Code Review (`code-review.md`)
Structured prompt for AI-powered code reviews focusing on:
- Security vulnerabilities
- Code quality and maintainability
- Test coverage
- Documentation
- Performance

### Security Audit (`security-audit.md`)
Comprehensive security analysis prompt covering:
- Authentication & authorization
- Input validation
- Dependencies
- Secrets management
- API security

### Bug Investigation (`bug-investigation.md`)
Systematic debugging approach for:
- Issue understanding
- Information gathering
- Problem isolation
- Root cause analysis
- Solution proposals

## Usage

These prompts can be used with:
- GitHub Copilot
- CyberAi agents
- SmartBrain AI integration
- Manual code reviews

## Integration

Reference these prompts in:
- `.github/copilot/agent.yaml`
- CI/CD workflows
- Pull request templates
- Issue templates
