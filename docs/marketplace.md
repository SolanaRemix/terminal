## Frequently Asked Questions

**Q: Does Terminal have access to my code?**
A: Terminal only reads public metadata and PR information. It never stores or transmits your source code.

**Q: Can I use Terminal with private repositories?**
A: Yes, with Pro or Enterprise tier subscriptions.

**Q: How are commands authenticated?**
A: Commands use your GitHub identity and permissions. You can only execute commands you have permission for.

**Q: Is there an API?**
A: Yes, Enterprise tier includes programmatic API access.

**Q: Can I self-host Terminal?**
A: Yes, self-hosting is available for Enterprise customers.
