# Terminal CLI

Command-line interface for Terminal automation.

## Usage

### TypeScript CLI

```bash
npx ts-node cli/terminal.ts help
npx ts-node cli/terminal.ts status
npx ts-node cli/terminal.ts scan
```

### PowerShell CLI

```powershell
./cli/terminal.ps1 help
./cli/terminal.ps1 status
./cli/terminal.ps1 scan
```

## Available Commands

- `help` - Display help and available commands
- `status` - Show repository status
- `scan` - Scan for issues or vulnerabilities
- `audit` - Run security and code audit
- `fix` - Apply automated fixes
- `deploy` - Deploy or publish

## CyberAi Ecosystem Commands
