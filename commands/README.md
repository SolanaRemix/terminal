# Terminal Command Implementations

This directory contains PowerShell implementations of Terminal commands.

## Available Commands

### Core Commands
- `scan.ps1` - Repository scanning and analysis
- `audit.ps1` - Security and code auditing
- `fix.ps1` - Automated fixes

## Usage

```powershell
# Run a scan
./commands/scan.ps1

# Run an audit
./commands/audit.ps1

# Apply fixes
./commands/fix.ps1
```

## TypeScript Implementations

See `/src/commands/` for TypeScript implementations of these commands that integrate with the GitHub webhook server.
